#EC-System
##简介
一个电子商务管理系统的课程设计

##开发方法
开发（数据库->DAO层->servlet->server层->前端）

##开发语言：
前端：jQuery-ui布局<br>
后台：java、JPA（数据库持久化）<br>
数据库：MySQL<br>
##开发工具：
myeclipse、Navicat可视化工具<br>
